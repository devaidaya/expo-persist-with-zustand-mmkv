# Zustand + MMKV Persist (Support de cours)

Ce dépôt sert de support au cours YouTube consacré à la persistance de données avec [Zustand](https://zustand-demo.pmnd.rs/) et [MMKV](https://github.com/mrousavy/react-native-mmkv) dans une application React Native construite avec [Expo](https://expo.dev).

## Objectif du cours

- Créer un store Zustand réactif pour gérer une todo-list.
- Persister l'état local avec MMKV afin de conserver les tâches après redémarrage.
- Mettre en place une UI moderne (Expo Router, React Native) pour illustrer l'utilisation du store persistant.

## Prérequis

- Node.js / Bun installé
- Expo CLI (`bunx expo` ou `npx expo`)

## Installation

```bash
bun install
```

## Lancer l'application

```bash
bunx expo start
```
